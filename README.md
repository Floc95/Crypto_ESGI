Cryptographie asymétrique
=========================

Autorité de certification : Interface pour utilisateur pour enregistrement
Autorité de validation : Autre utilisateur avec interface différente pour valider
Autorité d'enregistrement : Autre utilisateur avec interface différente pour enregistrer

Création en 4 étapes :

- Remplissage du formulaire + scan piède ID
- Validation de l'autorité de validation
- Enregistrement par l'autorité d'enregistrement
- Délivrance du certificat
